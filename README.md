# Transcriber

## Hugging Face token setup

Speaker diarization requires a Hugging Face token.

1. Copy `HF_TOKEN.example.txt` to `HF_TOKEN.txt`.
2. Replace the placeholder with your own token.
3. Keep `HF_TOKEN.txt` local (it is ignored by git).

Alternative: set `HF_TOKEN` in your environment instead of using a file.
